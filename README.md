Deze folder bevat een complete **modulaire skillset** voor AI-agents die websites beter vindbaar moeten maken in Google.

## Structuur
- `00-foundation/` basisprincipes en veiligheidsregels
- `01-discovery-indexing/` crawling/indexering skills
- `02-serp-research/` SERP analyse, keyword clustering, intent
- `03-topical-authority/` topical map, content roadmap
- `04-content-production/` writers + briefs + programmatic SEO
- `05-onpage-optimization/` titles/meta, headings, canonical
- `06-technical-seo/` CWV, mobile-first, speed, schema
- `07-authority-links/` linkbuilding & digital PR workflows
- `08-local-seo/` Google Business Profile, local landingpages
- `09-measurement-growth/` Search Console loops, dashboards

## Skill Format
Elke skill bevat:
- Doel
- Inputs
- Agent workflow
- Output format
- Checks / failure modes
- Prompt templates

## Gebruik (Antigravity)
Plaats deze folder in:
`/skills/google-seo/`

Agents kunnen skills automatisch oproepen op basis van taakbeschrijving, zoals:
- "Scan deze website op indexatieproblemen"
- "Maak een topical map voor een loodgieter in Amsterdam"
- "Genereer 50 programmatic SEO pagina's voor {service} + {stad}"

## Belangrijk
Deze skills zijn gemaakt voor **Nederlandse websites** (NL taal, NL SERP, lokale SEO).
