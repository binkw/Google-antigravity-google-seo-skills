# SEO principes

## Doel
SEO basis: hoe Google rankt en waarom je wint.

## Inputs
- Website niche
- Doelgroep
- Type content

## Agent Workflow
1. Identificeer zoekintentie
2. Bouw topical authority
3. Versterk vertrouwen (E-E-A-T)
4. Optimaliseer technische basis
5. Meet en verbeter via Search Console

## Output Format
Markdown rapport met prioriteiten + roadmap

## Checks / Failure Modes
Te veel focus op keywords zonder intent.
Geen interne links.
Geen autoriteit/backlinks.

## Prompt Template
```text
Analyseer deze website/niche en geef een SEO model met prioriteiten en stappenplan.
```
