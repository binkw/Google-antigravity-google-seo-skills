# Title & Meta Optimizer

## Doel
Optimaliseer titles en meta descriptions voor CTR.

## Inputs
- Pagina URL
- Keyword

## Agent Workflow
1. Analyseer huidige title/meta
2. Maak 5 varianten
3. Kies beste optie op CTR psychologie

## Output Format
Tabel met voorstellen

## Checks / Failure Modes
Te lang (>60 chars).
Te generiek.

## Prompt Template
```text
Maak betere title en meta description voor deze pagina: {url} keyword: {keyword}.
```
